# Bike Assistant & EMI Calculator 🏍️

A lightweight Python command-line utility that provides quick motorcycle specifications and calculates loan Equated Monthly Installments (EMI) across various standard tenures. 

This tool is perfect for buyers looking to quickly estimate the financial breakdown of a new two-wheeler purchase without navigating complex banking websites.

## 🚀 Features

* **Motorcycle Catalog:** View ex-showroom prices, engine capacity (cc), and mileage for popular bikes (e.g., Royal Enfield, KTM, Yamaha, Bajaj).
* **Dynamic EMI Calculation:** Input your desired down payment to instantly calculate loan amounts.
* **Tenure Breakdown:** Generates a comprehensive table showing standard loan tenures (3, 6, 9, 12, 36, and 60 months).
* **Financial Insights:** Calculates exact monthly EMIs, total interest payable, and the total amount to be paid at a standard 10% interest rate.
* **Tabular Output:** Uses the `pandas` library to output clean, easy-to-read data tables.

## 🛠️ Prerequisites

* Python 3.x
* Pandas library
